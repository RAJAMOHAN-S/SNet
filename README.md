## Requirements

- iOS 8.0+
- Xcode 10.1+

## Installation

### Using Cocoapods

```ruby
pod 'SNet', '~> 0.0.1' #Swift 4.2.1
```

``` swift
import SNet
```
#### Manually
1. Download and drop ```Source``` folder in your project.  
2. Congratulations!
